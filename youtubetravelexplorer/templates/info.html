How It Works

Overview

This application allows users to explore videos related to specific countries and cities on a map. Users can click on a country to view major cities and select a city to see related videos and current weather information.

APIs Used

  • YouTube Data API: Used to fetch videos related to the selected country or city.
  • Vimeo API: Used to fetch videos from Vimeo based on the search query.
  • Dailymotion API: Used to fetch videos from Dailymotion based on the search query.
  • Open-Meteo API: Used to fetch current weather data based on the selected city's coordinates.

Legal Considerations

All API usage complies with the respective terms of service. The YouTube Data API, Vimeo API, and Dailymotion API are used in accordance with their policies. Ensure that you have the necessary permissions and adhere to the usage limits set by these APIs.
